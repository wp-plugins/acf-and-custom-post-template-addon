== Description ==

This is an Advanced Custom Fields addon from Elliot and Custom Post Templates, it adds a new rule related to the CPT Plugin allowing you to add fields to this type of post.

== Installation ==

1. Upload `modern-browsing` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Rock and Roll!
